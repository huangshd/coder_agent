AGENT WORKFLOW OVERVIEW
================================================================================

1. CODER AGENT ⭐⭐⭐⭐⭐
   Workflow: Planner → N Workers (parallel) → K Checker (iterative)
   LLM Calls: 10-25 per request
   Cache: 15-60 GB
   SLO: < 30s
   Best For: Complex code generation with refinement

2. RAG AGENT ⭐⭐⭐
   Workflow: Embedding → Retrieval → Reranking → Context → Generation
   LLM Calls: 1 per request
   Cache: High potential (70% hit rate)
   SLO: < 5s
   Best For: Knowledge-based QA

3. MULTIMODAL AGENT ⭐⭐⭐⭐
   Workflow: Recognize → Preprocess (OCR/ASR) → Fuse → Understand
   LLM Calls: 1 per request
   Cache: Medium
   SLO: < 10s (< 20s for video)
   Best For: Image/audio understanding

4. CHATBOX AGENT ⭐⭐
   Workflow: Load Session → Build Context → Generate → Update
   LLM Calls: 1 per request
   Cache: High potential (80% hit rate with affinity)
   SLO: < 2s
   Best For: Conversational interactions
